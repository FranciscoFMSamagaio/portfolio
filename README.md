# Francisco Samagaio Portfolio

Static portfolio site built for GitHub Pages.

## Local preview

Open `index.html` directly in your browser, or run a simple local server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deploy on GitHub Pages

1. Push this repository to GitHub.
2. Open your repo settings: `Settings -> Pages`.
3. In `Build and deployment`, choose `Deploy from a branch`.
4. Select branch `main` and folder `/ (root)`.
5. Save. GitHub will publish the site at:
   `https://<your-username>.github.io/<repo-name>/`

## Content editing

- Main structure and text: `index.html`
- Visual style and responsiveness: `styles.css`
- Interactions and animations: `script.js`
